Welcome to the **System Design & Coding Interview Masterclass**. This repository is a curated collection of high-fidelity tracks designed to take you from foundational data structures to complex, large-scale system architectures. Whether you are preparing for a junior role or a staff engineering interview, this curriculum provides the depth and visual clarity needed to succeed.

---

## 🗺️ The Learning Path

We recommend following the tracks in the order below to build a solid foundation before moving into specialized domains.

### 00. [System Design Fundamentals](./00_System_Design_Fundamentals)
*The core building blocks of every digital system.*
- **Focus**: APIs, Load Balancers, Databases, Caching, and Networking.
- **Goal**: Understand how the internet and complex systems work beneath the surface.

### 01. [Coding Interviews (DSA)](./01_Coding_Interviews)
*Mastering patterns over problems.*
- **Focus**: Two Pointers, Sliding Window, Trees, Graphs, and Dynamic Programming.
- **Goal**: Develop the intuition to recognize algorithmic patterns in technical interviews.

### 02. [Object-Oriented Design (LLD)](./02_Object_Oriented_Design)
*Designing for maintainability and scalability.*
- **Focus**: SOLID principles, Design Patterns, and Class Diagrams.
- **Goal**: Build robust, extensible systems using industry-standard design patterns.

### 03. [System Design Interview (HLD)](./03_System_Design_Interview)
*Scaling to millions of users.*
- **Focus**: Distributed Systems, Rate Limiters, URL Shorteners, and Message Queues.
- **Goal**: Design end-to-end architectures that are reliable, scalable, and efficient.

### 04. [Mobile System Design](./04_Mobile_System_Design)
*Architecting for the palm of your hand.*
- **Focus**: Client-side storage, synchronization, networking, and offline support.
- **Goal**: Master mobile-specific architectural challenges (Android/iOS).

### 05. [Machine Learning System Design](./05_Machine_Learning_System_Design)
*Data-driven architectural patterns.*
- **Focus**: Feature engineering, model serving, and specialized ML pipelines.
- **Goal**: Design systems for visual search, recommendations, and ad-click prediction.

### 06. [Generative AI System Design](./06_Generative_AI_System_Design)
*The frontier of modern AI.*
- **Focus**: LLM infrastructure, RAG, Text-to-Video, and Diffusion models.
- **Goal**: Understand the architectural blueprints of tools like ChatGPT and Midjourney.

## 📚 How to Use

The curriculum is primarily delivered through **Jupyter Notebooks** (`.ipynb`). These notebooks are designed to be pedagogical, featuring:
- 🖼️ **High-Fidelity Visuals**: Complex concepts explained through detailed diagrams.
- 💬 **Interview Transcripts**: Simulated candidate-interviewer dialogues to mirror real sessions.
- 🏗️ **Architectural Blueprints**: Step-by-step breakdowns of system components.

### Running Locally
To get the best experience, we recommend running these notebooks in your local environment:
1. Clone the repository.
2. Install [Jupyter Lab](https://jupyter.org/install).
3. Open the desired track and start learning!
