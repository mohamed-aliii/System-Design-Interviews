# 05. Machine Learning System Design (MLSDI) Masterclass

Machine Learning System Design (MLSDI) covers the architecture of systems where machine learning models are the central components. It bridges the gap between pure data science and distributed systems engineering.

## 🎯 Learning Objectives
- Design end-to-end ML pipelines (Data collection → Processing → Training → Serving).
- Select appropriate models based on latency and throughput requirements.
- Implement feature engineering and embedding storage at scale.
- Understand online vs. offline evaluation and A/B testing.

## 🤖 The MLSDI Framework
1. **Clarify requirements**: Scale, latency, business goals (e.g., maximize CTR).
2. **Data & Features**: Feature engineering, data pipelines, labels.
3. **Model Selection**: Architectural trade-offs (e.g., Logistic Regression vs. Transformers).
4. **Evaluation**: Offline (Precision/Recall) vs. Online (A/B testing).
5. **Deployment & Serving**: Model hosting, retraining, and monitoring.
