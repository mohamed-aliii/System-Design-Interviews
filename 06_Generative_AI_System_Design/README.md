# 06. Generative AI System Design (GAISDI) Masterclass

Generative AI (GenAI) has revolutionized how we build applications. This track dives into the architectures behind Large Language Models (LLMs), Image Generation, and the infrastructure required to serve these computationally intensive models.

## 🎯 Learning Objectives
- Design systems using Retrieval-Augmented Generation (RAG).
- Understand LLM serving infrastructure and optimization (vLLM, quantization).
- Master Diffusion models and GANs for high-fidelity image synthesis.
- Architect multimodal GenAI systems (Text-to-Image, Text-to-Video).

## ⚡ GenAI Tech Stack
- **Foundation Models**: GPT-4, Claude, Llama 3, Stable Diffusion.
- **Vector Databases**: Pinecone, Milvus, Weaviate, Chroma.
- **Orchestration**: LangChain, LlamaIndex, Haystack.
- **Serving**: TGI (Text Generation Inference), vLLM, NVIDIA Triton.
- **Evaluation**: G-Eval, RAGAS, human-in-the-loop.
